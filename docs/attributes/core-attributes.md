## 📊 属性系统总览

### 属性分类

```
属性系统
├── 战斗属性（5 项）
│   ├── 功力 (martialPower)
│   ├── 外功 (externalSkill)
│   ├── 内功 (internalSkill)
│   ├── 轻功 (qinggong)
│   └── 体魄 (constitution)
│
├── 非战斗属性（6 项）
│   ├── 魅力 (charisma)
│   ├── 悟性 (comprehension)
│   ├── 侠义 (chivalry)
│   ├── 声望 (reputation)
│   ├── 人脉 (connections)
│   ├── 学识 (knowledge)
│   └── 财富 (wealth)
│
└── 隐藏属性（3 项，由天赋决定）
    ├── 武学潜力 (martialPotential)
    ├── 社交潜力 (socialPotential)
    └── 学习潜力 (learningPotential)
```

## ⚔️ 战斗属性详解

### 1. 功力 (martialPower)

**定义**: 综合武力水平，代表整体战斗能力  
**范围**: 0-100（可突破）  
**作用**: 
- 决定基础攻击力
- 影响可学习的武学等级
- 决定战斗中的威压效果
- 影响 NPC 对玩家武力的评价

**数值分段**:
```
0-10:    手无缚鸡之力（普通百姓）
11-30:   略有小成（入门弟子）
31-50:   驾轻就熟（江湖好手）
51-70:   融会贯通（一流高手）
71-90:   炉火纯青（顶尖高手）
91-100:  登峰造极（绝世高手）
100+:    返璞归真（传说境界）
```

**提升方式**:
- 修炼武功（外功/内功）
- 战斗历练
- 服用丹药
- 名师指点
- 天赋加成

**交互示例**:
```typescript
// 功力影响剧情分支
if (martialPower >= 80) {
  // 解锁"武林盟主"剧情线
  unlockPlotLine('wulin_master');
}

// 功力影响 NPC 态度
if (martialPower >= 60) {
  // 一流高手，NPC 敬畏
  npcAttitude += 20;
}
```

### 2. 外功 (externalSkill)

**定义**: 招式技巧，物理攻击能力  
**范围**: 0-100  
**作用**:
- 决定物理攻击力
- 影响招式威力和命中率
- 决定可学习的外功武学
- 影响战斗中的连击和暴击

**数值分段**:
```
0-10:    毫无章法
11-30:   初窥门径
31-50:   小有所成
51-70:   融会贯通
71-90:   出神入化
91-100:  登峰造极
```

**与功力的关系**:
```
实际攻击力 = 功力 * 0.6 + 外功 * 0.4
```

**提升方式**:
- 练习外功招式
- 实战演练
- 使用兵器
- 外功秘籍

**交互示例**:
```typescript
// 外功高于内功，偏向外功路线
if (externalSkill > internalSkill) {
  // 解锁外功专属事件
  triggerEvent('external_mastery_path');
}
```

---

### 3. 内功 (internalSkill)

**定义**: 内力修为，内力储备和运用能力  
**范围**: 0-100  
**作用**:
- 决定内力值（HP/MP）
- 影响内功威力
- 决定可学习的内功心法
- 影响疗伤和恢复速度
- 决定轻功发挥上限

**数值分段**:
```
0-10:    内力全无
11-30:   略有内力
31-50:   内力小成
51-70:   内力深厚
71-90:   内力磅礴
91-100:  内力如海
```

**与功力的关系**:
```
实际攻击力 = 功力 * 0.5 + 内功 * 0.5
内力上限 = 内功 * 10
```

**提升方式**:
- 打坐修炼
- 修炼内功心法
- 服用内丹
- 吸收天地灵气

**交互示例**:
```typescript
// 内功影响疗伤
healAmount = internalSkill * 2;

// 内功高可学习高级心法
if (internalSkill >= 70) {
  learnSecretManual('nine_yang_divine');
}
```

---

### 4. 轻功 (qinggong)

**定义**: 身法速度，闪避和移动能力  
**范围**: 0-100  
**作用**:
- 决定闪避率
- 影响先手概率
- 决定移动速度
- 影响追击和逃跑成功率
- 决定可学习的轻功身法

**数值分段**:
```
0-10:    步履蹒跚
11-30:   身法灵活
31-50:   来去如风
51-70:   飞檐走壁
71-90:   踏雪无痕
91-100:  缩地成寸
```

**战斗影响**:
```
闪避率 = 轻功 * 0.5%
先手率 = (轻功 - 对手轻功) * 1% + 50%
```

**提升方式**:
- 练习轻功身法
- 负重训练
- 悬崖峭壁修炼
- 特殊丹药

**交互示例**:
```typescript
// 轻功影响逃跑
if (qinggong > enemy.qinggong) {
  escapeSuccessRate = 80 + (qinggong - enemy.qinggong);
}

// 轻功高可触发特殊剧情
if (qinggong >= 70) {
  unlockLocation('forbidden_peak');
}
```

---

### 5. 体魄 (constitution)

**定义**: 身体素质，生命力和耐力  
**范围**: 0-100  
**作用**:
- 决定生命值上限（HP）
- 影响防御力
- 影响耐力和持续作战能力
- 决定抗毒、抗病能力
- 影响寿命

**数值分段**:
```
0-10:    体弱多病
11-30:   身体一般
31-50:   体格健壮
51-70:   铜皮铁骨
71-90:   金刚不坏
91-100:  肉身成圣
```

**战斗影响**:
```
生命值上限 = 体魄 * 10
防御力 = 体魄 * 0.5
```

**提升方式**:
- 锻炼身体
- 药浴
- 修炼硬功
- 服用炼体丹药

**交互示例**:
```typescript
// 体魄影响生存
maxHealth = constitution * 10;

// 体魄高可抵抗毒素
if (constitution >= 60) {
  poisonResistance = 50;
}
```

## 🎭 非战斗属性详解

### 6. 魅力 (charisma)

**定义**: 个人魅力，吸引力  
**范围**: 0-100  
**作用**:
- 影响 NPC 初始态度
- 影响说服成功率
- 影响社交事件触发
- 影响商店价格（砍价能力）

**社交影响**:
```
NPC 初始好感 = 魅力 * 0.5
说服成功率 = 魅力 * 0.8 + 其他因素
```

---

### 7. 悟性 (comprehension)

**定义**: 领悟能力，学习能力  
**范围**: 0-100  
**作用**:
- 影响武学学习速度
- 影响技能理解深度
- 影响突破瓶颈成功率
- 影响自创武学可能性

**学习影响**:
```
学习速度 = 1 + (悟性 * 0.01)
突破成功率 = 悟性 * 0.5%
```

---

### 8. 侠义 (chivalry)

**定义**: 道德倾向  
**范围**: -100~100  
**作用**:
- 正值：侠义道，负值：邪恶道
- 影响 NPC 态度（同阵营友好）
- 影响可接取的任务类型
- 影响结局分支

**阵营影响**:
```
if (chivalry > 50) 侠义道，NPC 尊敬
if (chivalry < -50) 邪恶道，NPC 畏惧
if (-50 <= chivalry <= 50) 中立
```

---

### 9. 声望 (reputation)

**定义**: 江湖名望  
**范围**: -1000~1000  
**作用**:
- 影响 NPC 态度
- 影响事件触发
- 影响可加入的势力
- 影响称号获取

**声望等级**:
```
-1000~-500:  臭名昭著
-499~-100:   声名狼藉
-99~99:      默默无闻
100~499:     小有名气
500~999:     名扬四海
1000+:       武林神话
```

## 🔗 属性关联规则

### 属性相互影响

```typescript
// 功力综合计算
martialPower = (externalSkill + internalSkill) / 2

// 战斗力综合评估
combatPower = martialPower * 0.6 + qinggong * 0.2 + constitution * 0.2

// 社交能力
socialAbility = charisma * 0.4 + connections * 0.3 + reputation * 0.3

// 综合能力
totalAbility = combatPower * 0.5 + socialAbility * 0.3 + knowledge * 0.2
```

### 属性与剧情交互

```typescript
// 示例 1：属性影响剧情分支
if (martialPower >= 80 && chivalry >= 50) {
  // 侠义高手路线
  triggerPlot('hero_path');
} else if (martialPower >= 80 && chivalry <= -50) {
  // 邪恶高手路线
  triggerPlot('villain_path');
}

// 示例 2：属性影响 NPC 态度
npcAttitude = base + (reputation * 0.02) + (chivalry * 0.01) + (charisma * 0.03);

// 示例 3：属性影响任务结果
if (connections >= 60) {
  // 人脉广，获得额外帮助
  questSuccessRate += 30;
}
```

## 📈 属性成长公式

### 基础成长

```typescript
// 属性成长 = 基础成长 + 天赋加成 + 其他加成

// 战斗属性成长
combatStatGrowth = baseGrowth * (1 + martialPotential * 0.01)

// 社交属性成长
socialStatGrowth = baseGrowth * (1 + socialPotential * 0.01)

knowledgeGrowth = baseGrowth * (1 + learningPotential * 0.01)
```

```typescript
// 外功修炼
externalGrowth = (trainingIntensity * 0.5) * (1 + comprehension * 0.01)
```

## 📝 开发指南

### 事件中使用属性

```json
{
  "id": "example_event",
  "conditions": [
    {
      "type": "expression",
      "expression": "martialPower >= 60 && chivalry >= 50"
    }
  ],
  "choices": [
    {
      "text": "展示武力（需功力≥60）",
      "condition": {
        "type": "expression",
        "expression": "martialPower >= 60"
      },
      "effects": [
        {
          "type": "stat_modify",
          "target": "reputation",
          "value": 10,
          "operator": "add"
        }
      ]
    }
  ]
}
```

### 属性平衡原则

1. **避免单一属性决定一切** - 多属性综合评判
2. **保持成长曲线平滑** - 避免暴涨暴跌
3. **提供多种发展路径** - 不同属性组合都能成功
4. **属性价值动态调整** - 不同场景不同属性重要性不同
