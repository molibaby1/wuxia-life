## 技术实现亮点

### 1. 事件结构设计
```json
{
  "id": "唯一标识",
  "version": "版本号",
  "category": "事件类别",
  "priority": "优先级",
  "weight": "权重",
  "ageRange": {"min": 最小年龄，"max": 最大年龄},
  "triggers": ["触发器数组"],
  "conditions": ["条件数组"],
  "content": {"标题", "描述", "文本"},
  "eventType": "choice/auto/ending",
  "choices": ["选择项数组"],
  "autoEffects": ["效果数组"],
  "requirements": {"属性需求", "标记需求"}
}
```

### 2. 触发机制
- **age_reach**: 年龄触发
- **random**: 概率触发
- **flag_based**: 标记触发

### 3. 条件系统
- **属性条件**: chivalry >= 20
- **标记条件**: flags.buddhist_fate == true
- **复合条件**: 多个条件组合

### 4. 效果系统
- **stat_modify**: 属性修改
- **flag_set**: 设置标记
- **flag_unset**: 移除标记

### 5. 路线分支
- 佛门：禅宗/武僧/双修/俗家
- 正道：武当/峨眉/华山
