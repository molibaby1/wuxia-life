# Artifact and Output Convention

This project distinguishes source assets from generated runtime artifacts.

## Generated Outputs (Do Not Commit)

- `public/reports/*.html`
- `public/reports/*.json`

These files are generated by simulation and verification workflows. They are volatile and should not be committed.

## Tracked Output Index

- `public/reports/manifest.json`

`manifest.json` is the only tracked report artifact and acts as a stable index for report discovery.

## Rule of Thumb

If an output can be deterministically regenerated from source and scripts, treat it as an artifact and keep it out of source control unless explicitly designated as tracked.
