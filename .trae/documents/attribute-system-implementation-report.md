## 完成情况

### ✅ 第一步：侠义值专属事件（10 个）

**文件**: `src/data/lines/chivalry-events.json`

#### 高侠义事件（5 个）
1. **正道联盟邀请** - 侠义≥80 触发，加入正道联盟
2. **太极剑法传承** - 侠义≥85+ 武力≥90，学习正道绝学
3. **村民感恩** - 侠义≥70，受助村民感谢
4. **英雄救美** - 侠义≥75+ 武力≥70，选择分支事件
5. **侠义传奇** - 侠义≥90+ 声望≥500，名声传遍江湖

#### 低侠义事件（5 个）
1. **魔教招募** - 侠义≤-50，加入魔教选择
2. **吸星大法** - 侠义≤-60+ 武力≥90，学习邪派武功
3. **拦路抢劫** - 侠义≤-40，道德抉择
4. **背叛盟友** - 侠义≤-50，黑吃黑选择
5. **武林公敌** - 侠义≤-80，被正道追杀

**属性价值体现**:
- ✅ 侠义值决定可触发的事件
- ✅ 影响事件分支选项
- ✅ 决定可学习的武功类型
- ✅ 改变 NPC 态度和剧情走向

### ✅ 第二步：金钱消耗事件（12 个）

**文件**: `src/data/lines/money-events.json`

#### 金钱门槛事件（4 个）
1. **武林拍卖会** - 金钱≥500 触发，竞拍秘籍/神兵/灵药
2. **地下黑市** - 金钱≥1000 触发，购买禁术/毒药/情报
3. **投资机会** - 金钱≥2000 触发，投资店铺获得被动收入
4. **门派募捐** - 金钱≥100 触发，捐赠获得声望

#### 金钱消耗选项（8 个）
1. **身受重伤** - 花费 150/50 两，不同治疗效果
2. **武馆训练** - 花费 200/80/20 两，不同训练效果
3. **建立名声** - 花费 300/100/30 两，不同规模宴席
4. **山贼拦路** - 花费 100 两破财消灾 vs 武力解决
5. **神秘商人** - 花费 500/150 两，购买不同秘籍
6. **茶馆情报** - 花费 100/30 两，购买不同情报
7. **武器升级** - 花费 400/150/30 两，不同等级升级
8. **赎人事件** - 花费 300 两赎金 vs 武力营救

**属性价值体现**:
- ✅ 金钱决定事件触发资格
- ✅ 金钱解锁更好的选项
- ✅ 金钱改变事件走向
- ✅ 金钱购买特殊服务和物品

## 典型事件示例

### 示例 1：侠义值影响分支

```json
{
  "title": "山贼劫村",
  "choices": [
    {"text": "挺身而出", "effect": "侠义 +15，声望 +20"},
    {"text": "视而不见", "effect": "侠义 -5"},
    {"text": "趁火打劫", "req": "侠义≤-20", "effect": "金钱 +100，侠义 -30"}
  ]
}
```

### 示例 2：金钱门槛和消耗

```json
{
  "title": "武林拍卖会",
  "condition": "金钱≥500",
  "choices": [
    {"text": "买秘籍", "cost": -600, "effect": "武力 +28"},
    {"text": "买神兵", "cost": -400, "effect": "武力 +18"},
    {"text": "买灵药", "cost": -250, "effect": "体质 +12"}
  ]
}
```

### 示例 3：声望解锁功能

```json
{
  "title": "开宗立派",
  "condition": "声望≥500 AND 武力≥100",
  "choices": [
    {"text": "创立门派", "cost": -1000, "effect": "获得门派"},
    {"text": "时机未到"}
  ]
}
```
