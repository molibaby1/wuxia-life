# 老年事件扩展报告

## 📋 任务概述

为 50-70 岁年龄段添加更多老年事件，丰富游戏后期体验。

## ✅ 完成内容

### 传承线（5 个事件）

1. **50 岁：整理武学** (`elderly_organize_martial`)
   - 描述：年事已高，你决定整理毕生所学，以免失传
   - 选择：
     - 系统整理，著书立说 (学识 +20, 声望 +20)
     - 口传心授，传给弟子
   - 状态：✅ 已存在并优化

2. **55 岁：归隐田园** (`elderly_retirement`)
   - 描述：你决定远离江湖纷争，归隐田园
   - 选择：
     - 归隐山林，不问世事 (体魄 +10)
     - 移居小镇，安度晚年
   - 状态：✅ 已存在并优化

3. **60 岁：著书立说** (`elderly_write_memoirs`)
   - 描述：你决定将自己的武学心得和人生感悟记录下来
   - 选择：
     - 详实记录，流传后世 (学识 +15, 声望 +15)
     - 简单记录，留给子女
   - 状态：✅ 已存在并优化

4. **65 岁：衣钵传人** (`elderly_succession`)
   - 描述：你的传人已经学成，可以独当一面了
   - 选择：
     - 举行传承仪式 (声望 +30)
     - 简单传授，顺其自然
   - 状态：✅ 已存在并优化

5. **70 岁：圆满人生** (`elderly_life_review` + `elderly_final_ending`)
   - 描述：在生命的尽头，你回顾自己的一生
   - 选择：
     - 无怨无悔，心满意足 (圆满结局)
     - 仍有遗憾，希望来生
   - 状态：✅ 已存在并优化

### 江湖传说线（4 个事件）

1. **52 岁：江湖传说** (`elderly_legend_begins`) ⭐ **新增**
   - 描述：你的名声已经传遍江湖，成为人们口中的传奇

   - 状态：✅ 新增成功

2. **58 岁：故人重逢** (`elderly_reunion`) ⭐ **新增**

   - 选择：

   - 触发条件：老友未去世
   - 状态：✅ 新增成功

3. **62 岁：最后的心愿** (`elderly_final_wish`) ⭐ **新增**
   - 描述：你感到时日无多，想要完成最后的心愿
   - 选择：
     - 重游故地，缅怀往昔
     - 了却恩怨，放下一切 (侠义 +10)
   - 状态：✅ 新增成功

4. **68 岁：临终托付** (`elderly_entrustment`) ⭐ **新增**
   - 描述：在生命的最后时刻，你将家人和弟子叫到床前
   - 选择：

   - 状态：✅ 新增成功

## 🔧 技术实现

### 1. 事件格式标准化

所有事件均使用统一的标准化格式：
- `choices` 字段在顶层（不在 `content` 内）
- `autoEffects` 用于自动事件
- 效果类型使用 `stat_modify`、`flag_set`、`special` 等标准类型
- 属性名称使用 `money`、`comprehension`、`connections`、`reputation`

### 2. 触发条件设计

- **年龄范围**：每个事件都有合理的年龄范围（如 50-65 岁）
- **属性要求**：部分事件需要达到一定属性（如著书立说需要学识 ≥ 50）
- **Flag 依赖**：部分事件需要前置条件（如衣钵传人需要先有传人）

### 3. 效果系统

- **属性修改**：使用 `stat_modify` 类型，支持 `add`、`set` 等操作符
- **Flag 设置**：使用 `flag_set` 类型，标记事件完成状态
- **特殊效果**：使用 `special` 类型处理 `end_game` 等特殊情况
