# 多阶段爱情系统说明文档

**创建时间**: 2026-03-14  
**文件位置**: `src/data/lines/love-mature.json`  
**设计理念**: 爱情不分年龄，每个阶段都有独特的情感体验

---

## 📋 系统概述

传统的武侠游戏往往将爱情线限定在青少年时期，一旦错过就要终老一生。这不符合真实的人生体验。

**多阶段爱情系统**的核心理念：
- ✅ **爱情不分年龄**：每个年龄段都有适合的爱情故事
- ✅ **错过不是终点**：年轻时错过的缘分，中年、老年仍有重逢的可能
- ✅ **情感多元化**：包含初恋、成熟之恋、黄昏恋、婚外情、重逢旧爱等多种情感形态
- ✅ **选择有代价**：每个选择都会带来不同的后果和影响

---

## 🎭 爱情阶段详解

### 1️⃣ 成熟之恋（30-45 岁）

**适合人群**: 年轻时专注于武道/事业，错过初恋的玩家

**事件流程**:
```
love_mature_first (30 岁触发)
  ↓ 接受感情
love_mature_bond (31 岁发展)
  ↓ 相知相守
→ 获得属性加成：魅力 +8, 侠义 +5
```

**特色**:
- 成熟的爱情更加稳重
- 不需要轰轰烈烈，只需相知相守
- 对方理解并支持你的江湖路

**属性影响**:
- ✅ 接受：魅力 +5, 侠义 +3, 魅力 +8 (后续)
- ❌ 拒绝：武力 +3 (专心武道)

---

### 2️⃣ 再续前缘（35-55 岁）

**适合人群**: 曾经爱情失败（flags.love_ended）的玩家

**前置条件**: 
- `flags.love_ended == true` (之前的爱情已结束)

**选择分支**:
```
love_second_chance
  ├─ 勇敢接受 → 新恋情开始，魅力 +6, 体质 +5
  └─ 犹豫不决 → 侠义 -2, 错失良机
```

**特色**:
- 爱情不会只来一次
- 给对方也给自己第二次机会
- 更加珍惜来之不易的缘分

---

### 3️⃣ 婚外情缘（35-60 岁）⭐

**适合人群**: 已有伴侣但魅力值高的玩家

**前置条件**:
- `flags.love_started == true` 或 `flags.love_married == true`
- `player.charisma >= 50`

**道德抉择**:
```
love_affair_temptation
  ├─ 坚守底线 → 侠义 +15, 魅力 +5, 现任好感 +10 ✅
  ├─ 陷入情网 → 侠义 -20, 魅力 +8, 现任好感 -30, 情人好感 +25 ⚠️
  └─ 向现任坦白 → 侠义 +5, 现任好感 -15
```

**后续发展** (如果选择陷入情网):
```
love_affair_consequence (36 岁触发)
  ├─ 结束婚外情，回归家庭
  │   → 侠义 +5, 声望 -15, 情人好感 -50, 现任好感 -10
  │
  └─ 选择情人，离开家庭
      → 侠义 -15, 声望 -30, 金钱 -200, 离婚，前任好感 -100
```

**特色**:
- 真实的道德困境
- 每个选择都有代价
- 后果系统完善

**设计说明**:
婚外情是一个有争议但真实存在的情感现象。游戏不鼓励这种行为（侠义值会大幅下降），但允许玩家做出自己的选择并承担后果。

---

### 4️⃣ 黄昏情缘（55-75 岁）

**适合人群**: 老年仍未结婚的玩家

**前置条件**:
- `flags.love_married != true`
- `flags.love_elderly_started != true`

**事件流程**:
```
love_elderly_companion (55 岁触发)
  ↓ 珍惜缘分
love_elderly_memories (60 岁发展)
  ↓ 执子之手
→ 获得属性加成：魅力 +10, 体质 +8, 健康 +20, 健康 +25 (后续)
```

**特色**:
- 爱情不分年龄，真心永不褪色
- 相濡以沫的感情弥足珍贵
- 一起回忆往昔，谈论江湖往事

**属性影响**:
- 接受：魅力 +10, 体质 +8, 健康 +20, 健康 +25 (后续), 声望 +10
- 做朋友：魅力 +5, 健康 +10

**文案亮点**:
> "最浪漫的事，就是和你一起慢慢变老。"

---

### 5️⃣ 重逢旧爱（45-70 岁）

**适合人群**: 年轻时错过初恋的玩家

**前置条件**:
- `flags.love_first_meet_pass == true` (当年选择了"只是远远一望")

**选择分支**:
```
love_reunion_old
  ├─ 再续前缘 → 新恋情，魅力 +8, 侠义 +5
  ├─ 做老朋友 → 魅力 +5, 声望 +5
  └─ 相忘于江湖 → 侠义 +3
```

**特色**:
- 有些缘分，注定要等待
- 岁月在彼此脸上留下了痕迹，但心动依然清晰
- 给年轻时的遗憾一个圆满的结局

## 🔧 技术实现

### Flag 系统

**新增 Flags**:
- `love_mature_started` - 成熟之恋开始
- `love_mature_bonded` - 成熟之恋发展
- `love_second_chance` - 再续前缘
- `love_affair` - 有婚外情
- `love_affair_resisted` - 拒绝了婚外情诱惑
- `love_affair_confessed` - 坦白了婚外情
- `love_affair_exposed` - 婚外情暴露
- `love_divorced` - 离婚
- `love_elderly_started` - 黄昏恋开始
- `love_elderly_friend` - 老年知己
- `love_elderly_memories` - 老年回忆
- `love_reunion_old` - 重逢旧爱
- `love_reunion_friend` - 重逢后做朋友
- `love_reunion_ignored` - 重逢后相忘

### 条件检查示例

```json
{
  "id": "love_mature_first",
  "conditions": [
    {
      "type": "expression",
      "expression": "!flags.has(\"love_started\") && !flags.has(\"love_mature_started\")"
    }
  ]
}
```

**说明**: 确保玩家当前没有其他恋情，才能触发新的爱情线

## 🎯 设计亮点

### 1. 包容性强
- 不限制爱情发生的年龄
- 不限制爱情的形式
- 尊重玩家的选择权

### 2. 真实性高
- 包含婚外情等真实情感困境
- 每个选择都有相应后果
- 符合人性的复杂性

### 3. 文案优美
- "成熟的爱情不需要轰轰烈烈，只需相知相守"
- "最浪漫的事，就是和你一起慢慢变老"
- "有些缘分，注定要等待"

### 4. 平衡性好
- 不同阶段的爱情有不同的属性加成
- 年轻时的爱情加成魅力，老年的爱情加成健康
- 婚外情虽然诱人，但代价巨大

## 📝 使用建议

### 给玩家的建议

1. **年轻时**：如果遇到喜欢的人，勇敢追求
2. **错过后**：不要灰心，30 岁后还有成熟之恋
3. **中年时**：面对诱惑要慎重，代价可能很大
4. **老年时**：依然可以相信爱情，相濡以沫最珍贵

### 给开发者的建议

1. **扩展更多爱情线**：可以添加师徒恋、异国恋等
2. **增加互动事件**：与恋人的日常互动、共同任务
3. **完善后果系统**：婚外情的后果可以更复杂（如子女关系）
4. **添加成就系统**：如"白头偕老"（与同一恋人到老）、"情圣"（多段恋情）等

## 🎉 总结

**多阶段爱情系统**的核心理念：

> **爱情不是年轻人的专利，每个年龄段都有属于自己的浪漫。**

- 年轻时错过，不代表一生遗憾
- 中年时的爱情更加成熟稳重
- 老年时的相伴更加珍贵
- 每个选择都有意义，每段感情都值得珍惜

这个系统让游戏更加贴近真实人生，让玩家在江湖闯荡的同时，也能体验完整的情感历程。
