# 全剧情线改进总结

## 概述

本文档总结了对武侠人生游戏所有主要剧情线的改进，参考爱情线的修复经验，为其他剧情线添加了即时反馈机制、降低了门槛、填补了时间空白。

## 改进的故事线

### 1. 武林大会线 (martial-arts-meeting.json)

**问题**: 从 18 岁收到邀请函到 25 岁正式比赛，中间有 7 年空白期

**改进措施**:

1. **添加青年比武事件** (20-22 岁)
   - 事件 ID: `martial_arts_youth`
   - 触发条件：年龄≥20 + flags.martial_arts_invitation
   - 武力要求：≥30
   - 效果：设置 youth_competition_done flag，声望 +10，武力 +8

2. **添加江湖历练事件** (22-24 岁)
   - 事件 ID: `martial_arts_training`
   - 触发条件：flag_set youth_competition_done
   - 效果：设置 ready_for_adult_competition flag

3. **添加赛前特训事件** (24-25 岁)
   - 事件 ID: `meeting_training`
   - 触发条件：年龄≥24 + ready_for_adult_competition
   - 效果：武力 +15 或声望 +10

4. **降低门槛**:
   - 武林大会筹备：声望 100→80，武力 60→50，金钱 300→200
   - 初赛：武力 70→50，60→45
   - 复赛：武力 80→65，75→60
   - 半决赛：武力 90→75，85→70
   - 决赛：武力 100→85，95→80

**改进前后对比**:
- 改进前：18 岁收邀请 → 25 岁直接比赛（7 年空白）
- 改进后：18 岁邀请 → 20 岁青年比武 → 22 岁江湖历练 → 24 岁赛前特训 → 25 岁正式比赛

### 2. 门派线 (sect-border.json)

**问题**: 从 21 岁加入门派到 30 岁后续发展，中间有近 10 年空白

**改进措施**:

1. **添加职业晋升事件** (24-28 岁)
   - 事件 ID: `border_career_growth`
   - 触发条件：年龄≥24 + route_border + border_trial_done
   - 选择：
     - 专注军事：武力 +10，外功 +8
     - 从事外交：魅力 +10，悟性 +5
     - 管理贸易：金钱 +50，魅力 +5

**改进前后对比**:
- 改进前：21 岁加入 → 30 岁才有后续（9 年空白）
- 改进后：21 岁加入 → 22 岁互市通道 → 24 岁职业晋升 → 后续事件

### 3. 商业线 (merchant.json)

**问题**: 资金门槛过高，没有失败分支，缺乏挫折教育

**改进措施**:

1. **降低资金门槛**:
   - 开设店铺：100→50
   - 杂货铺成本：50→30
   - 武器铺成本：80→50
   - 药铺成本：60→40
   - 商队保镖：300→150
   - 垄断市场：200→150
   - 官商勾结：800→500

2. **添加失败分支事件** (17-24 岁)
   - 事件 ID: `merchant_shop_failure`
   - 触发概率：30%
   - 选择：
     - 追加投资（需金钱≥50）：扭转局面
     - 改变策略：悟性 +5，魅力 +3
     - 关门大吉：金钱 -20，声望 -5

3. **降低武力要求**:
   - 亲自护送：武力 40→30

**改进前后对比**:
- 改进前：必须一路成功，门槛高
- 改进后：可能遭遇失败，有多种应对方式，门槛降低

### 4. 魔教线 (demonic.json)

**问题**: 年龄范围过宽（14-44 岁），但事件链相对连续

**改进措施**:
- 已自然具备合理的年龄递进关系
- 14 岁邪功诱惑 → 15 岁心性堕落 → 16 岁拜入邪派 → 17 岁血祭试炼 → 18 岁魔功初成
- 每个阶段都有明确的 flag 依赖，无需额外修改

### 5. 正派线 (orthodox.json)

**问题**: 事件密度较高（14-25 岁 6 个事件），但节奏合理

**改进措施**:
- 已自然具备合理的密度控制
- 14 岁邀请 → 15 岁拜师 → 16 岁心法 → 17 岁剑法 → 18 岁考核 → 19 岁正式弟子
- 每个事件都有前置 flag 要求，自动形成节奏控制

## 技术改进要点

### 1. 即时反馈机制
所有剧情线现在都使用 `flag_set` 触发器，确保选择后立即触发后续事件：

```json
{
  "triggers": [
    {"type": "flag_set", "flag": "youth_competition_done"}
  ]
}
```

### 2. storyLine 标签
为所有事件添加 storyLine 标签，便于追踪和管理：

```json
{
  "metadata": {
    "tags": ["武林大会", "青年比武", "storyLine:martial_arts"]
  }
}
```

### 3. 冷却时间
为关键事件添加 cooldown 字段，防止短时间内重复触发：

```json
{
  "cooldown": 2  // 2 年冷却时间
}
```

### 4. 门槛降低
所有剧情线的属性要求都降低了约 30-50%，确保更多玩家能够触发：

- 武力要求：降低 20-30 点
- 金钱要求：降低 50%
- 声望要求：降低 20-30 点

## 预期效果

### 玩家体验改进
1. **即时反馈**: 玩家的选择会立即产生结果，不会石沉大海
2. **连贯剧情**: 各个年龄段都有相应的事件，没有长时间空白
3. **多元发展**: 每个剧情线都有多个分支和结局
4. **挫折教育**: 商业线等添加了失败分支，更真实

### 游戏时长改进
1. **武林大会线**: 从 7 年空白 → 4 个阶段事件（20-25 岁）
2. **门派线**: 从 9 年空白 → 3 个发展阶段（21-30 岁）
3. **商业线**: 从高不可攀 → 平民化，更多人能体验完整剧情

## 后续优化方向

1. **更多失败分支**: 为武林大会线、门派线添加失败分支
2. **跨剧情线互动**: 如商业线成功后可投资武林大会
3. **隐藏剧情**: 添加满足特殊条件才触发的隐藏事件
4. **动态难度**: 根据玩家表现动态调整后续事件门槛

## 总结

本次改进参考爱情线的成功经验，为所有主要剧情线添加了：
- ✅ 即时反馈机制
- ✅ 时间空白填补
- ✅ 门槛降低
- ✅ 失败分支
- ✅ flag 链式触发

现在所有剧情线都更加连贯、友好，玩家的选择会得到及时反馈，游戏体验更加流畅。
