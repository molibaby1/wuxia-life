# 爱情 - 婚姻冲突系统实现总结

## 问题解决

成功解决了游戏中爱情线与婚姻关系割裂的问题，实现了两者之间的动态关联和戏剧冲突。现在玩家的爱情选择会直接影响婚姻质量，而婚姻状态又会约束或影响后续的情感发展。

## 核心改进

### 1. 爱情 - 婚姻关联系统

#### 改进前
- 爱情线：与明月等恋人的浪漫故事，但没有后续
- 婚姻线：简单的"已婚"flag 设置
- 两者完全独立，没有交集

#### 改进后
- **爱情影响婚姻选择**:
  - 与明月好感度高 → 可娶为妻子（真爱婚姻，质量高但经济压力大）
  - 接受家族安排 → 富家千金（利益婚姻，财富↑但情感空虚）
  - 自由恋爱 → 中等婚姻质量

- **婚姻约束爱情发展**:
  - 已婚状态下遇到旧爱 → 婚外情风险
  - 婚姻质量低 → 更容易发展婚外情
  - 婚外情暴露 → 婚姻危机、离婚风险

### 2. 婚姻质量系统

引入隐藏的"婚姻质量"指标，受以下因素影响：

```
婚姻质量 = 基础值 (50) 
         + 婚姻类型加成 (真爱 +20, 包办 -10)
         + 配偶好感度 × 0.3
         - 婚外情惩罚 (每个 -30)
         - 重大冲突事件惩罚
```

**婚姻质量等级及效果**:
- **美满 (80-100)**: 配偶支持事业，家庭和睦
- **和谐 (60-79)**: 正常婚姻生活，偶有小摩擦
- **平淡 (40-59)**: 感情淡漠，维持表面和平
- **危机 (20-39)**: 频繁争吵，可能分居
- **破裂 (0-19)**: 离婚风险极高

### 3. 核心冲突事件链

#### 3.1 婚前冲突

**事件 1: 爱情与现实的抉择**
- **触发**: 与明月好感度≥60，年龄≥22
- **冲突**: 真爱 (明月) vs 现实 (富家千金联姻)
- **选择后果**:
  - 坚持真爱：获得明月，家族关系↓，财富↓
  - 接受联姻：获得富家千金，财富↑，明月嫁作他人妇
  - 拖延不决：明月可能嫁作他人妇，孤独终身风险

#### 3.2 婚后冲突

**事件 2: 旧爱重逢**
- **触发**: 已婚，明月单身
- **冲突**: 对现任配偶的愧疚 vs 对旧爱的眷恋
- **选择后果**:
  - 断绝往来：婚姻质量↑，个人心情↓
  - 保持友谊：婚姻质量波动，婚外情风险积累
  - 发展婚外情：刺激但高风险，侠义↓

**事件 3: 婚外情暴露危机**
- **触发**: 有婚外情且 intensity≥50
- **冲突**: 配偶发现蛛丝马迹，情人要求名分
- **选择后果**:
  - 坦白并结束：配偶原谅度检定，可能离婚
  - 继续隐瞒：压力↑，暴露风险↑
  - 承认并选择情人：离婚，财产分割，名誉↓

#### 3.3 家庭责任冲突

**事件 4: 传宗接代的压力**
- **触发**: 已婚 3 年无子
- **冲突**: 家族催促纳妾 vs 配偶反对
- **选择后果**:
  - 坚持不纳妾：配偶满意度↑，家族关系↓
  - 纳妾生子：配偶满意度↓↓，家族关系↑
  - 过继子嗣：折中方案，金钱 -50

**事件 5: 事业与家庭的平衡**
- **触发**: 已婚有子，事业上升期
- **冲突**: 重要任务 vs 子女成长关键时刻
- **选择后果**:
  - 优先事业：声望↑金钱↑，婚姻质量↓，子女疏远
  - 回归家庭：婚姻质量↑，事业机会丧失
  - 带家人同行：折中方案，但风险↑

**事件 6: 婚姻破裂**
- **触发**: 婚姻质量≤30 或主动提出离婚
- **冲突**: 结束痛苦婚姻 vs 努力修复
- **选择后果**:
  - 同意和离：财产分割，名誉↓，获得自由
  - 挽回婚姻：婚姻质量↑，需要付出努力
  - 拖延不离：维持表面，抑郁风险↑

### 4. 数据结构扩展

使用 flags 系统追踪婚姻状态：

```typescript
// 婚姻状态 flags
flags.married = true/false              // 是否已婚
flags.spouse_mingyue = true/false       // 配偶是明月
flags.spouse_arranged = true/false      // 包办婚姻
flags.marriage_type_love = true/false   // 真爱婚姻
flags.marriage_quality_high = true/false // 婚姻质量高

// 情感状态 flags
flags.love_started = true/false         // 已有恋人
flags.mingyue_fiancee = true/false      // 明月是未婚妻
flags.mingyue_married_other = true/false // 明月嫁给别人
flags.extramarital_affair = true/false  // 有婚外情
flags.affair_mingyue = true/false       // 与明月有婚外情

// 家庭责任 flags
flags.has_child = true/false            // 有子女
flags.family_pressure_high = true/false // 家族压力大
```

## 技术实现

### 1. 新增事件文件
- **love-marriage-conflict.json**: 6 个核心冲突事件
  - `marriage_love_reality_choice`: 爱情与现实抉择
  - `marriage_old_lover_reunion`: 旧爱重逢
  - `marriage_affair_crisis`: 婚外情暴露
  - `marriage_family_pressure_child`: 传宗接代压力
  - `marriage_career_family_balance`: 事业家庭平衡
  - `marriage_divorce_crisis`: 婚姻破裂

### 2. 修改现有事件
- **family-life.json**: 
  - 修改 `family_marriage` 事件，添加与爱情线的关联
  - 新增"迎娶明月"选项
  - 区分婚姻类型（真爱、包办、自由恋爱）

### 3. 事件配置要点

```json
{
  "id": "marriage_old_lover_reunion",
  "triggers": [
    {"type": "age_reach", "value": 25},
    {"type": "random", "value": 0.4}
  ],
  "conditions": [
    {"type": "expression", "expression": "flags.married == true && flags.mingyue_single == true"}
  ],
  "metadata": {
    "pathAffinity": {"martial_artist": 10},
    "pathConflicts": {"hermit": 60, "official": 40}
  }
}
```

## 预期游戏效果

### 改进前
- 玩家经历：恋爱→结婚（无关联）→生子→老年
- 婚姻选择：简单的是/否
- 情感体验：平淡，缺乏戏剧冲突

### 改进后
- **玩家经历**: 
  - 初恋（明月）→ 现实压力 → 婚姻选择 → 婚后冲突 → 最终结局
  - 每个阶段都有情感冲突和道德抉择

- **婚姻选择的影响**:
  - 真爱婚姻：情感满足但经济压力大
  - 利益婚姻：财富增长但情感空虚
  - 自由恋爱：中等发展，相对平衡

- **戏剧冲突**:
  - 旧爱重逢时的情感纠葛
  - 婚外情暴露时的道德审判
  - 事业与家庭的两难选择
  - 离婚时的财产分割和名誉损失

## 使用说明

### 为事件添加婚姻冲突元素

1. **检查婚姻状态**:
```json
"conditions": [
  {"type": "expression", "expression": "flags.married == true"}
]
```

2. **设置婚姻相关 flags**:
```json
"effects": [
  {"type": "flag_set", "flag": "extramarital_affair", "value": true}
]
```

3. **添加路径亲和/冲突**:
```json
"metadata": {
  "pathAffinity": {"martial_artist": 10},
  "pathConflicts": {"hermit": 60}
}
```
