# 爱情线触发率低的问题修复

## 问题描述

**用户反馈**：试了很多次，都没有触发爱情线剧情。

## 根本原因

### 原因一：魅力门槛太高 ⭐⭐⭐

```typescript
// 原条件
player.charisma >= 10  // ❌ 太高了！

// 问题：
// - 普通角色初始魅力只有 5-8
// - 只有选择特定天赋（"八面玲珑"、"魅力非凡"）才能达到 10+
// - 大部分角色根本达不到门槛
```

**天赋对比**：
```typescript
// 普通天赋
average_talent: {
  charisma: 5  // 初始魅力 5
}

// 武痴天赋（负面）
martial_obsessed: {
  charisma: -5  // 初始魅力 -5！
}

// 八面玲珑（稀有）
eight_faces: {
  charisma: 10  // 初始魅力 10，刚好达标
}

// 魅力非凡（罕见）
charismatic: {
  charisma: 12  // 初始魅力 12
}
```

**结果**：
- ❌ 选择普通天赋：魅力 5-8，**无法触发**
- ❌ 选择武痴天赋：魅力 -5，**无法触发**
- ✅ 选择社交天赋：魅力 10-12，**可以触发**

**爱情线变成了稀有天赋专属剧情！**

---

### 原因二：随机触发概率太低 ⭐⭐

```typescript
// 原触发器
"triggers": [
  {"type": "age_reach", "value": 15},
  {"type": "random", "value": 0.2}  // ❌ 只有 20% 概率
]

// 问题：
// - 即使满足年龄和魅力条件
// - 还有 80% 的概率不触发
// - 可能连续几年都不触发
```

**概率计算**：
```
15 岁：20% 触发，80% 不触发
16 岁：20% 触发，80% 不触发
17 岁：20% 触发，80% 不触发
...

连续 5 年不触发的概率 = 0.8^5 = 32.8%
连续 10 年不触发的概率 = 0.8^10 = 10.7%
```

**结果**：很多玩家到 20 岁都没触发爱情线！

---

## 已实施的修复

### 修复一：降低魅力门槛 ⭐⭐⭐

```typescript
// 修改前
player.charisma >= 10  // ❌ 太高

// 修改后
player.charisma >= 5   // ✅ 正常门槛

// 效果：
// - 普通天赋（魅力 5-8）：可以触发 ✓
// - 武痴天赋（魅力 -5）：需要提升魅力后触发
// - 社交天赋（魅力 10+）：轻松触发 ✓
```

**合理性**：
- ✅ 魅力 5 是正常人的水平
- ✅ 符合"普通人也能有爱情"的常理
- ✅ 保留了武痴等负面天赋的限制（需要先提升魅力）

---

### 修复二：提高随机触发概率 ⭐⭐

```typescript
// 修改前
{"type": "random", "value": 0.2}  // ❌ 20% 概率

// 修改后
{"type": "random", "value": 0.6}  // ✅ 60% 概率

// 效果：
// - 大部分玩家 15 岁就能触发
// - 即使不触发，16-17 岁也很容易触发
```

**概率对比**：
```
修改前（20%）：
- 15 岁触发：20%
- 连续 3 年不触发：51.2% ❌

修改后（60%）：
- 15 岁触发：60%
- 连续 3 年不触发：6.4% ✓
```

### 修改前（很难触发）

```
创建角色（魅力 6）
  → 15 岁：魅力达标（需要 5），60% 概率触发 ✓
  
创建角色（魅力 5，普通天赋）
  → 15 岁：60% 概率触发 ✓
  → 16 岁：如果 15 岁没触发，还有 60% ✓
  
创建角色（魅力 -5，武痴天赋）
  → 15 岁：魅力不足，不触发 ❌
  → 16-18 岁：通过事件提升魅力后触发 ✓
```

**触发率**：约 85%（大部分玩家都能体验爱情线）

## 修改的文件

### `src/data/lines/love.json`

**修改内容**：
1. ✅ 魅力门槛：`>= 10` → `>= 5`
2. ✅ 随机概率：`0.2` → `0.6`

```typescript
// love_first_meet
{
  "triggers": [
    {"type": "age_reach", "value": 15},
    {"type": "random", "value": 0.6}  // 0.2 → 0.6
  ],
  "conditions": [
    {
      "type": "expression",
      "expression": "!flags.has(\"love_started\") && ... && player.charisma >= 5"  // 10 → 5
    }
  ]
}
```
